# typora.theme.deepash
基于typora官方ash修改的主题

![image-20211221152606465](assets/image-20211221152606465.png)

## 测试版本

Mac 10.14 typora 0.11.16

## 安装

- 下载压缩包或者release资源并解压
- 找到对应的主题文件夹,将deepash.css和deepash文件夹拷贝到主题文件夹下
- 选择Deepash主题
